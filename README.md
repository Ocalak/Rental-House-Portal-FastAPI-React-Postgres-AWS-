# World-Wide-Wohnung

```
git clone https://github.com/TechLabs-Dortmund/world-wide-wohnung.git

cd backend

python -m venv env 

source env\bin\activate 

pip install -r .\requirements.txt

uvicorn main:app --reload     #start server 

visit  https://127.0.0.1:8000/docs ```
